Mobile Navigation Test Page

Test the mobile nav drawer at different viewport sizes.

Instructions

  1. Resize browser to mobile width (<768px) or use DevTools mobile emulation
  2. Click the hamburger menu icon in the header
  3. Mobile nav drawer should slide in from the right
  4. Body scroll should be blocked (can't scroll page)
  5. Click the X or click outside to close
