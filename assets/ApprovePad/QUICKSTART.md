# Quickstart

## Install AutoHotkey v2

- Download and install AutoHotkey v2 from the official site.

## Run Approve Pad

- Double-click: `src\Approve-Pad.ahk`

## Common prompts supported

- Yes / No
- 1 / 2 / 3
- Approve All / Always

## If it doesn't send keys

- Click into the target window once so it becomes active.
- If the target window is elevated (Admin), run Approve Pad as Admin.
