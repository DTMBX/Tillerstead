# Build Phase: A Complete Guide to Code-Aware Bathroom & Shower Construction in New Jersey

This page gathers **all Build Phase guides in one place**. It is designed as the public-facing hub for homeowners who want to understand how a bathroom or shower is properly planned, built, inspected, and verified in New Jersey.

Each guide below focuses on **one phase of construction**, written in plain language and ordered the same way a real project unfolds.

---

## How to Use This Page

* New to remodeling? Start at the top and read in order.
* Researching a specific issue? Jump directly to the topic that matters to you.
* Comparing contractors? Use these guides to understand *what should be happening behind the scenes*.

This is not a DIY manual. It is an **education and alignment resource** so expectations are clear before work begins.

## Why This Build Phase Matters

Most bathroom problems are not caused by tile or fixtures. They are the result of **missed decisions, rushed sequencing, or unverified systems** earlier in the build.

These guides exist to:

* Set clear expectations
* Reduce surprises during construction
* Help homeowners partner with professionals
